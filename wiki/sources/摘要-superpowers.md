# 摘要：Superpowers

## 素材来源
- **原始文件**: `raw/01-articles/superpowers.md`
- **主题**: obra/superpowers 智能体技能框架

## 核心主旨
Superpowers 是一套跨平台的 AI 编程智能体技能框架与软件开发方法论，强调测试驱动开发（TDD）、系统性流程和子代理驱动开发，支持 Claude Code、Cursor、GitHub Copilot、OpenAI Codex 等多种工具。

## 关键知识点

1. **核心理念**: TDD（始终先写测试）、系统性优于临时性、降低复杂度、证据胜于断言

2. **基本工作流程**:
   1. 头脑风暴（Brainstorming）
   2. 使用 Git 工作树（Git Worktree）
   3. 编写计划（Write Plan）
   4. 子代理驱动开发（Subagent-Driven Development）或执行计划
   5. 测试驱动开发（Test-Driven Development）
   6. 请求代码审查（Request Code Review）
   7. 完成开发分支（Finish Dev Branch）

3. **技能库**:
   - 测试: TDD（红-绿-重构）
   - 调试: 系统性调试、完成前验证
   - 协作: 头脑风暴、编写/执行计划、并行代理、代码审查、Git 工作树、子代理驱动开发
   - 元技能: 编写技能、使用 Superpowers

4. **跨平台支持**: Claude Code（官方市场）、Cursor（插件市场）、GitHub Copilot CLI、OpenAI Codex CLI/Gemini CLI/OpenCode
